# homebrew-tap

Homebrew formulae for [yongjip](https://github.com/yongjip)'s tools.

```sh
brew install yongjip/tap/mergetrain
```

| Formula | Description |
|---|---|
| [`mergetrain`](https://github.com/yongjip/mergetrain) | Local-first merge queue for parallel coding-agent worktrees |

Formulae track PyPI releases automatically (daily bump workflow).
